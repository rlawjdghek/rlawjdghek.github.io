du -sh : 현재 디렉토리 하위까지 포함한 총 용량

du -sh * : 현재 디렉토리에 있는 파일 / 폴더를 각각 표시해줌

du -s : 현재 디렉토리의 용량

du -h : 현재디렉토리에 있는 모든 파일/폴더 용량 나열
