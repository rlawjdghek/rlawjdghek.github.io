**cuda 다운로드 받을 때 보면 cuda_11.2.2_46.32.33**처럼 뒤에 nvidia driver 버전이 있다. 이걸 맞춰서 깔아야 한다. 지금 글쓰는 시점에서는 A100과 3090이 주력이므로 cuda 11.2와 NVIDIA_DRIVER 460,
cudnn 8.1.1을 설치하였다.


# 삭제
```bash
1. sudo apt-get remove --purge '^nvidia-.*' 
2. sudo apt-get autoremove --purge 'cuda*'
3. sudo apt-get autoremove --purge 'cuda*'
4. sudo rm -rf /usr/local/cuda*

```

# 설치
```bash
1. sudo apt-get update
2. sudo apt-get install build-essential dkms
3. sudo apt-get install freeglut3 freeglut3-dev libxi-dev libxmu-dev
```
4. [nvidia-driver](https://www.nvidia.co.kr/Download/index.aspx?lang=kr)에서 nvidia-driver 설치. 이거 깔면 nvidia-smi가 된다. 따라서 nvidia-smi에 있는 cuda version이 제대로 된것이 
아니라는 것은 먼저 nvidia-driver가 먼저 깔리기 때문에 여기서 뜨는 버전은 드라이버와 호환되는 쿠다를 나타낸다.
5. [cuda](https://developer.nvidia.com/cuda-toolkit-archive)에서 cuda를 설치한다. 이 때 nvidia-driver는 1.에서 깔았으므로 체크를 해제해야한다.
6. 5.를 깔면 중간 메세지에 "please make sure ~"라고 되어있는데 이것은 path를 추가하라는 말이다.
```bash
7. export PATH=$PATH:/usr/local/cuda-<쿠다 버전>/bin
8. export CUDADIR=/usr/local/cuda-<쿠다 버전> 7., 8.을 입력하면 nvcc -V가 된다.
``` 
9. [cudnn](https://developer.nvidia.com/rdp/cudnn-archive)에서 cudnn 다운로드. 맨위에 있는 tgz파일을 다운로드한다. 아래는 deb파일. 
10. tar xvzf 로 압축을 풀면 cuda 폴더가 생긴다. 이것을 쿠다가 설치되어있는 곳으로 복사해야 된다. 아래까지 해야 cudnn을 사용하는 것이고, cudnn 버전체크가 된다.
```bash
11. sudo cp ./cuda/include/* /usr/local/cuda-<쿠다 버전>/include  
12. sudo cp -P ./cuda/lib64/* /usr/local/cuda-<쿠다 버전>/lib64
13. sudo chmod a+r /usr/local/cuda-<쿠다 버전>/lib64/libcudnn* 
``` 


# 버전 체크
1. nvidia-driver -> nvidia-smi 
2. cuda -> nvcc -V
3. cudnn 8이전 -> cat /usr/local/cuda/include/cudnn.h | grep CUDNN_MAJOR -A 2 
4. cudnn 8이후 -> cat /usr/local/cuda/include/cudnn_version.h | grep CUDNN_MAJOR -A 2
