# 설치
1. sudo apt-get update
2. sudo apt-get install build-essential dkms
3. sudo apt-get install freeglut3 freeglut3-dev libxi-dev libxmu-dev
4. https://developer.nvidia.com/cuda-toolkit-archive
에서 버전 설정하고 추천 명령어 입력.
5. export PATH=$PATH:/usr/local/cuda-10.2/bin
6. export CUDADIR=/usr/local/cuda-10.2
7. export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/local/cuda-10.2/lib64
8. https://developer.nvidia.com/rdp/cudnn-archive 가서 cuda 버전에 맞는 cudnn 설치
9. sudo dpkg -i 로 cudnn 패키지 풀기

7까지 하면 cuda, nvidia-driver가 설치되어 nvidia-smi가 되고,
9까지하면 cudnn도 설치 되어 nvcc -V가 된다.

# 삭제
1. sudo apt-get purge nvidia* 
2. sudo apt-get autoremove 
3. sudo apt-get autoclean 
4. sudo rm -rf /usr/local/cuda*
